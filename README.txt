Nom du projet : Visualisation audio
Ce projet est un site web simple permettant de visualiser les fréquences de l'audio.
Il a été réalisé dans le cadre de défi de la nuit de l'info. 
Fonctionnalités
-Ajouter un morceau musicale
-Visualise les audio sous formes des formes géométrique. 
-Interface simple et responsive. 
Technologies utilisées: 
HTML5
CSS3
JavaScript Vanilla 
L’API Web Audio API (native au navigateur)
L’API Canvas 2D
